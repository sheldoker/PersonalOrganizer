# 📒 Personal Organizer

> Персональный кроссплатформенный органайзер для Windows, Android и Wear OS.

---

# 🎯 Цель проекта

Создать современный персональный органайзер с возможностью хранения заметок, управления задачами, ведения календаря и синхронизации данных между всеми устройствами пользователя.

Проект разрабатывается как долгосрочный учебный проект с использованием современных технологий платформы .NET и лучших практик разработки.

---

# 🖥 Поддерживаемые платформы

## Версия 1.0

- Windows (WPF)

## Версия 2.0

- Android (.NET MAUI)

## Версия 3.0

- Wear OS

# 📋 Техническое задание

## 📝 Заметки

### Основной функционал

- Создание заметок
- Редактирование
- Удаление
- Поиск по названию
- Поиск по содержимому
- Архив
- Корзина
- Закрепление
- Избранное

### Дополнительные возможности

- Категории
- Цветные категории
- Теги
- Приоритет

---

## 📂 Категории

Каждая категория должна содержать:

- Название
- Цвет
- Описание (опционально)

Примеры категорий:

- 🟥 Работа
- 🟦 Учёба
- 🟩 Дом
- 🟨 Покупки

Категории используются в:

- Заметках
- Календаре
- ToDo

---

## 🏷 Теги

Пользователь может создавать неограниченное количество тегов.

Примеры:

- #работа
- #университет
- #важно
- #экзамен

Одна заметка может иметь несколько тегов.

---

## ⭐ Приоритет

Каждый объект приложения может иметь приоритет.

Уровни:

- 🔴 Высокий
- 🟡 Средний
- 🟢 Низкий

Используется для:

- Заметок
- Событий
- ToDo

---

## 📅 Календарь

Типы событий:

- События
- Встречи
- Дни рождения
- Праздники

Функции:

- Создание
- Редактирование
- Удаление
- Повторяющиеся события
- Напоминания
- Категории
- Приоритет

---

## ✅ ToDo

Функции:

- Создание списков задач
- Несколько списков
- Чекбоксы
- Дедлайны
- Категории
- Приоритет
- Фильтрация

---

## 👤 Аккаунт

Функции:

- Регистрация
- Авторизация
- Выход
- Редактирование профиля

---

## ☁ Синхронизация

После входа в аккаунт пользователь получает доступ ко всем данным на любом устройстве.

Синхронизируются:

- Заметки
- Категории
- Теги
- Календарь
- ToDo
- Настройки

Поддерживаемые устройства:

- Windows
- Android
- Wear OS

---

## ⚙ Настройки

Темы оформления:

- Светлая
- Тёмная
- Системная

Дополнительно:

- Резервные копии
- Импорт
- Экспорт

---

## 🔍 Поиск

Поиск осуществляется по:

- Названию
- Содержимому
- Категориям
- Тегам

---

## 🗑 Корзина

Удалённые элементы не удаляются сразу.

Возможности:

- Восстановление
- Полное удаление
- Автоматическая очистка (в будущем)

---

## 📦 Архив

Позволяет хранить старые заметки без удаления.

Функции:

- Перемещение в архив
- Восстановление

---

## ❤️ Избранное

Любую заметку можно добавить в избранное для быстрого доступа.

---

## 📌 Закрепление

Закреплённые заметки всегда отображаются первыми.
